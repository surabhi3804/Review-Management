# Review Management System

A smart system to analyze customer reviews using sentiment analysis and root cause detection.

## Features
- Sentiment Analysis on customer reviews
- Root Cause Analyzer to identify real issues
- Real-time alerts for negative feedback
- Competitor comparison dashboard

## Tech Stack
- Python
- React
- Node js
- AI logic
- Web Scraping (Zomato reviews)

## Use Case
Helps businesses quickly identify service problems and improve customer satisfaction.
